# 📊 데이터마이닝 (Data Mining)

---

## 📚 수업 개요
본 과목은 데이터 처리와 분석의 기본 개념부터 고급 기법까지 포괄적으로 학습하며, 실제 데이터 분석 역량을 기르는 것을 목표로 한다.

데이터 전처리, 데이터 분석, 모델링, 평가 등 데이터마이닝의 전 과정을 Python과 관련 라이브러리(Pandas, NumPy, Matplotlib 등)를 활용하여 실습 중심으로 학습한다.

또한 SQL 기반 데이터 처리와 데이터 시각화 기술을 포함하여 데이터 핸들링 능력을 심화한다.

다양한 예제 데이터 기반 실습과 edX 강의를 활용한 학습을 병행하며, 프로젝트를 통해 윤리적 데이터 수집, 텍스트 데이터 분석, SQL 기반 데이터 처리 등의 실무 사례를 경험한다.

더 나아가 GPT와 같은 대형 언어 모델(LLM)을 데이터 분석에 접목하여 데이터마이닝과 인공지능 기술의 융합 능력을 키우는 것을 목표로 한다.

---

## 🧠 주요 학습 내용
- 데이터 전처리 및 정제
- 탐색적 데이터 분석(EDA)
- 데이터 모델링 및 평가
- SQL 기반 데이터 처리
- 데이터 시각화
- 텍스트 데이터 분석
- 윤리적 데이터 수집 및 크롤링
- LLM(GPT 등)을 활용한 데이터 분석

- 프로젝트 기반 실전 경험 습득
